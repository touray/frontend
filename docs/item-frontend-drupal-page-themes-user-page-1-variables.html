6.6.3.1.1 Variables

User Page 1 component variables. To add/edit component variables, use scss/_variables.scss.

  • If true, display the site header. Default: false

    $user-page-header

  • If true, display the site footer. Default: false

    $user-page-footer

  • Defines the body background for the logged out version of the user page. Default: $light-gray

    $user-page-background

  • Defines the background for the main block on the logged out version of the user page. Default: $white

    $user-page-block-background

  • Defines the padding for the main block on the logged out version of the user page. Default: $global-padding

    $user-page-block-padding

  • Defines the shadow for the main block on the logged out version of the user page. Default: 0 0 25px rgba(0, 0, 0, 0.07)

    $user-page-block-shadow

  • Defines the max-width for the main block on the logged out version of the user page. Default: rem-calc(400)

    $user-page-block-width

  • Defines the title for the main block on the logged out version of the user page. Default: "Sign In"

    $user-page-block-title

  • Defines the title size for the main block on the logged out version of the user page. Default: precent-size(22)

    $user-page-block-title-size

  • Defines the title font weight for the main block on the logged out version of the user page. Default: $global-weight-bold

    $user-page-block-title-weight

  • Defines the border color for the input on the logged out version of the user page. Default: $light-gray

    $user-page-input-border-color

  • Defines the focus border color for the input on the logged out version of the user page. Default: $primary-color

    $user-page-input-border-color-focus

  • Defines the label font size on the logged out version of the user page. Default: $small-font-size

    $user-page-label-size

  • Defines the label font color on the logged out version of the user page. Default: $black

    $user-page-label-color

  • Defines the input description font size on the logged out version of the user page. Default: $small-font-size

    $user-page-input-desc-size

  • Defines the input description font color on the logged out version of the user page. Default: $medium-gray

    $user-page-input-desc-color

  • Defines the input description focus font color on the logged out version of the user page. Default: $dark-gray

    $user-page-input-desc-color-focus
