4.2.1.1 Building Semantically

XY grid CSS is generated with a powerful set of Sass mixins, which you can use in your own code to build a semantic grid. For more info, see https://foundation.zurb.com/sites/docs/xy-grid.html#building-semantically.

The markup and classes below are for examples purposes. They should not be used in your project.
