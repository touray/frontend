6.7.1.3.1 Variables

Modal 1 component variables. To add/edit component variables, use scss/_variables.scss.

  • Modal border. Default: 0

    $modal-border

  • Modal radius. Default: $global-radius

    $modal-radius

  • Modal font size. Default: $global-font-size

    $modal-size

  • Modal shadow. Default: $modal-shadow

    $modal-shadow

  • Modal title font size. Default: $global-font-size

    $modal-title-size

  • Modal title font color. Default: $white

    $modal-title-color

  • Modal title font shadow. Default: none

    $modal-title-shadow

  • Modal title font weight. Default: $global-weight-bold

    $modal-title-weight

  • Modal header background. Default: $black

    $modal-header-background

  • Modal close link color. Default: $white

    $modal-close-color

  • Modal close link font size. Default: rem-calc(13)

    $modal-close-size
