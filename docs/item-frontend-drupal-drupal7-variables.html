6.5.1.1 Variables

Drupal 7 component variables. To add/edit component variables, use scss/_variables.scss.

  • Defines the site message margin. Default: rem-calc($global-margin 0)

    $message-margin

  • Defines the site message padding. Default: $global-padding

    $message-padding

  • Defines the site message border. Default: rem-calc(1) solid

    $message-border

  • Defines the status message background. Default: rgba($success-color, .5)

    $message-status-background

  • Defines the status message border. Default: rem-calc(1) solid darken($success-color, 50%)

    $message-status-border

  • Defines the status message color. Default: darken($success-color, 50%)

    $message-status-color

  • Defines the warning message background. Default: rgba($warning-color, .5)

    $message-warning-background

  • Defines the warning message border. Default: rem-calc(1) solid darken($warning-color, 50%)

    $message-warning-border

  • Defines the warning message color. Default: darken($warning-color, 50%)

    $message-warning-color

  • Defines the error message background. Default: rgba($alert-color, .5)

    $message-error-background

  • Defines the error message border. Default: rem-calc(1) solid darken($alert-color, 50%)

    $message-error-border

  • Defines the error message color. Default: darken($alert-color, 50%)

    $message-error-color

  • Defines the progress bar background. Default: $progress-background

    $progress-bar-background

  • Defines the progress bar border. Default: rem-calc(1) solid $medium-gray

    $progress-bar-border

  • Defines the percentage text float. Default: right

    $progress-bar-percentage-float

  • Defines the progress bar margin. Default: rem-calc(0 0 $global-margin 0)

    $progress-bar-margin

  • Defines the progress bar radius. Default: rem-calc($progress-radius)

    $progress-bar-radius

  • Defines the progress filled background. Default: $progress-meter-background

    $progress-filled-background

  • Defines the progress height. Default: $progress-height

    $progress-height

  • Defines the progress weight. Default: $global-weight-bold

    $progress-weight
