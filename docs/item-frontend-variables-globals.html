6.6.1 Globals

These are the default global variables. To add/edit, use scss/_variables.scss.

  • Defines the application's default max width. Default: rem-calc(1200)

    $global-width

  • Defines the application's default margin. Default: 1rem

    $global-margin

  • Defines the application's default padding. Default: 1rem

    $global-padding

  • Defines the application's default radius. Default: 0

    $global-radius

  • Defines the application's default normal weight. Default: normal

    $global-weight-normal

  • Defines the application's default bold weight. Default: bold

    $global-weight-bold

  • Defines the application's default line height. Default: 1.5

    $global-lineheight

  • Defines the body font family. Default: 'Helvetica Neue', Helvetica, Roboto, Arial, sans-serif

    $body-font-family
