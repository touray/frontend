3.5 #frontend.variables Variables

The variables file for Frontend Build components. Foundation is used by default in Frontend Build, so uses it's settings to define many of the default Frontend Build component variables. It's best to edit those variables from Foundation's scss/framework/foundation/_settings.scss. All other variables and custom variables should be defined in the scss/_variables.scss file.

Globals

  • $global-width: $global-width (Foundation setting)
  • $global-margin: $global-margin (Foundation setting)
  • $global-padding: $global-padding (Foundation setting)
  • $global-radius: $global-radius (Foundation setting)
  • $global-weight-bold: $global-weight-bold (Foundation setting)
Colors
  • $primary-color: $primary-color (Foundation setting)
  • $alert-color: $alert-color (Foundation setting)
  • $success-color: $success-color (Foundation setting)
  • $warning-color: $warning-color (Foundation setting)
  • $dark-gray: $dark-gray (Foundation setting)
  • $medium-gray: $medium-gray (Foundation setting)
  • $light-gray: $light-gray (Foundation setting)
Progress Bar
  • $progress-background: $progress-background (Foundation setting)
  • $progress-margin-bottom: $progress-margin-bottom (Foundation setting)
  • $progress-radius: $progress-radius (Foundation setting)
  • $progress-meter-background: $progress-meter-background (Foundation setting)
  • $progress-height: $progress-height (Foundation setting)
CSS Grid
  • $css-grid-columns: (small: 2, medium: $grid-columns) (Foundation setting)
  • $css-grid-column-gaps: $grid-margin-gutters (Foundation setting)
  • $css-grid-row-gaps: (small: 0, medium: 0)
