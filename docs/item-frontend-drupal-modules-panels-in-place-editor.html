6.7.3.2 Panels In-Place Editor

Contains default Panels In-Place Editor module styles. To add/edit component variables, use scss/_variables.scss.

  • Sticky control container height. Default: $black

    $control-container-height

  • Sticky control container background. Default: $black

    $control-container-background

  • Sticky control container box shadow. Default: none

    $control-container-shadow

  • Sticky control container padding. Default: rem-calc(10 0)

    $control-container-padding

  • Sticky control button background. Default: $button-background

    $control-button-background

  • Sticky control button hover background. Default: $button-background-hover

    $control-button-background-hover

  • Sticky control button color. Default: $button-color

    $control-button-color

  • Sticky control button hover color. Default: $button-color-hover

    $control-button-color-hover

  • Sticky control button font family. Default: $button-font-family

    $control-button-font-family

  • Sticky control button padding. Default: $button-padding

    $control-button-padding

  • Sticky control button font sizes. Default: map-get($button-sizes, small)

    $control-button-size

  • Sticky control button weight. Default: $global-weight-bold

    $control-button-weight
