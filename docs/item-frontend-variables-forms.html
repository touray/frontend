6.6.5 Forms

These are the default options for form styling. To add/edit, use scss/_variables.scss.

  • Defines the font size for form labels. Default: rem-calc(14)

    $form-label-font-size

  • Defines the form input bottom margin. Default: rem-calc(16)

    $form-spacing

  • Defines the form label font weight. Default: $global-weight-bold

    $form-label-font-weight

  • Defines the form input radius. Default: $global-radius

    $input-radius

  • Defines the select radius. Default: $input-radius

    $select-radius
